# SmartGov Assistant (Web) - Demo Project

This is a full demo web project for **SmartGov Assistant** built with **Vite + React + Tailwind CSS**. It includes sample data, PDF export, language toggle, templates UI and Firebase-ready placeholders.

## Quick start

1. Install dependencies
```bash
npm install
```

2. Run locally
```bash
npm run dev
```

3. Open the app
Visit the URL printed in terminal (usually http://localhost:5173).

## Firebase
- The project includes `src/firebaseConfig.js` with **placeholders**. Paste your Firebase web app config if you want Auth/Firestore features.

## Notes
- `src/data/documents.json` contains the sample checklist data.
- This project is ready to run offline (Firebase optional).
